# Contributing

Este repositorio es un proyecto academico y de portafolio.

Antes de proponer cambios:

1. Ejecuta `uv sync --python 3.11 --extra dev`.
2. Ejecuta `make lint`.
3. Ejecuta `make test`.
4. No incluyas datos grandes, `.venv`, caches ni reportes generados.

Todo texto publico debe mantener el encuadre etico: priorizacion de revision humana,
explicacion auditable y trazabilidad.
