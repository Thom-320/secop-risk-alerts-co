# Scoring explicable

El score es una priorizacion de revision humana. Combina:

- componente de anomalia;
- desviacion frente a pares;
- reglas explicables;
- confianza de datos;
- razones auditables.

La salida no es una conclusion juridica ni fiscal. Sirve para ordenar la revision y
mostrar por que un proceso merece atencion humana.
